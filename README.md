# TaskGenie - AI-Powered Task Management

An intelligent task management system with NLP-powered task breakdown and context-aware classification.

## Tech Stack
- **Backend**: Node.js, Express, MongoDB
- **Frontend**: React, Tailwind CSS
- **NLP Service**: Python, Flask, Hugging Face Transformers
- **Authentication**: JWT
